# SpatialNeurogenomicsEngine

Analyze spatial navigation circuits including grid cells, place cells, and hippocampal theta oscillations.

## Features

- Grid cell detection and gridness score computation
- Place cell field mapping across environments
- Hippocampal theta oscillation phase analysis
- Spatial memory encoding and retrieval modeling
- Multi-environment generalization assessment

## Results

100 envs × 50 neurons; Grid cells=64%; Mean gridness=0.41; Place cells=80%; Theta MRL=0.675

## Usage

```bash
pip install numpy scipy matplotlib
python spatial_neurogenomics_engine.py
```
